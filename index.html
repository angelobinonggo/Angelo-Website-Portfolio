Pet Center System Prototype

Figma Website: Pet Center System Prototype Overview

The Pet Center System Prototype, designed in Figma, is a user-friendly web-based platform that streamlines pet care services such as appointment scheduling, pet record management, and service listings. With a modern and responsive UI, the prototype offers seamless navigation for both pet owners and employees, featuring an intuitive booking system, detailed pet profiles, and an employee dashboard for efficient management. Interactive elements simulate real user interactions, ensuring a clear visualization of the user journey before development. This prototype serves as a blueprint for a fully functional system, enhancing the efficiency of pet care businesses while providing a smooth digital experience for customers.

Java NetBeans

Metro Animal Clinic System

Java NetBeans: Metro Animal Clinic System Features

The Metro Animal Clinic System, developed in Java NetBeans, is a comprehensive desktop application designed to streamline veterinary clinic operations. This system facilitates efficient management of pet records, appointment scheduling, medical history tracking, and billing processes. With a user-friendly interface and a well-structured database, it ensures seamless access to critical information for veterinarians and staff. Built with object-oriented programming (OOP) principles, the system enhances workflow efficiency by automating routine tasks, reducing paperwork, and improving service delivery. Its robust architecture allows for easy scalability and future enhancements, making it a reliable solution for modern veterinary clinics.

VS Code

USTP Website Clone

VS Code: USTP Website Clone Development.

The USTP Website Clone, developed using VS Code, is a meticulously crafted replica of the official University of Science and Technology of Southern Philippines (USTP) website. This project showcases advanced web development skills, incorporating HTML, CSS, and JavaScript to recreate the website's structure, design, and interactivity. The clone serves as a practice model for understanding modern web design principles, responsive layouts, and user experience optimization. By replicating key features such as navigation menus, content sections, and interactive elements, the project highlights proficiency in front-end development while demonstrating attention to detail and accuracy in web replication.
